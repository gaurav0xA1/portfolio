Welcome to Sim8085® Help Pages ...

There are some alternative ways to display the help pages, please choose one of them which suits to you:

  1. Normal view: Standart browser window is displayed; required resolution is 1024x768.
  2. Simplified view: In this mode, browser window has only the page area. But, required resolution is again 1024x768.
  3. Float mode: With this mode, menu applet can be shown in a seperate window. So, required resolution is less than others: 800x600.
  4. Simplified Float mode: This mode is combination of Float mode and Simplified mode.

While loading the page, the window mode is selected automatically according to the current screen resolution. If you wanna change this or if something is wrong, then click the buttons above.
