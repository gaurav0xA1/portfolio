---------- About Sim8085 ----------

Sim8085 is an integrated program that collects three features in one:

First, it is an editor. You can use this editor like the "NotePad" program of Windows. (may be more than that ! ;-) )
Second, it is a compiler. You can compile (or say syntax checking) your code from inside the editor.
Third, it is a debugger. You can debug your code, just like your code runs on a 8085 Cpu. Here, you can observe the registers or any memory content. You can also change the values or even the code itself within debugger !

As mentioned before, in the 'Copyright Notice', this program is written for educational purposes only. It is written because there is no such a program that runs under Windows, and flexible, for using as an assistant with our "Microprocessor Architecture and Programming" course. It is thought that, Sim8085 will be enough according to lesson content, hence it is designed NOT to be so complicated.

When designing both the Sim8085 and this help file, the course book "Microprocessor Architecture, Programming, and Applications with the 8085/8085A" is taken as a reference. (Ramesh S. Gaonkar, second edition)
