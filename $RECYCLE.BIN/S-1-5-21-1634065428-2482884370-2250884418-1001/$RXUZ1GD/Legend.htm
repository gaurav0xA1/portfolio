The abbreviations used in the description of the instruction set:

Reg = 8085 registers
mem = memory
R = register
Rs = source register
Rd = destination register
Rp = register pair
M = memory
[...] = memory contents of ...
xx = random information
cont = content
sub = subtract

Flags:

S = sign
Z = zero
AC = auxiliary carry
P = parity
CY = carry
